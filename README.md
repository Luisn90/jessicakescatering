# Jessicakes Catering — Website

Sitio web oficial de **Jessicakes Catering and Events Inc.** — Tampa, Florida.

## Stack
- HTML5 + CSS3 + Vanilla JS
- Sin dependencias externas (fuentes desde Google Fonts)
- SVG logos inline
- Totalmente responsive

## Características
- Pantalla de carga animada con logo
- Cursor personalizado con inercia
- Parallax en hero
- Scroll reveal por sección
- Marquee animado
- Formulario de contacto con labels flotantes
- Botón flotante de WhatsApp
- Navegación con glassmorphism al hacer scroll

## Deploy con GitHub Pages
1. Ve a **Settings → Pages**
2. Source: **Deploy from branch**
3. Branch: **main** / root
4. Guarda — en ~1 min estará live en `https://tuusuario.github.io/jessicakes-website`
